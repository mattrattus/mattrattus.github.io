What is admin-helper?

It’s very simple list of commands, which are used to check daily, what is happening on your server.

At the moment, it supports servers that are based on Debian.

What commands does it have?

last, journalctl, systemctl, netstat, fail2ban-client, apt update

Practical use

First, you need to download the script.
My recommended method is to download it directly from my GitHub:

git clone https://github.com/mattrattus/admin-helper.git

When the script is already on our disk, now all you have to do is change the permission to execute:

chmod +x admin.sh

Then all you have to do is run the script:

./admin.sh

And that’s it.
